# 21MOON SHORTS HOOKS V1

## 1. Propósito
Este documento reúne hooks, aperturas y líneas de arranque pensadas para piezas verticales cortas de 21-Moon.

Está diseñado para contenido orgánico y paid en TikTok, Reels, YouTube Shorts, trailers cortos, anuncios verticales y piezas breves de awareness. La intención no es llenar videos con frases lindas, sino abrirlos con tensión, presencia y deseo de una forma coherente con la marca.

## 2. Qué debe lograr un hook de 21-Moon
Un hook de 21-Moon debe:
- frenar el scroll en los primeros segundos;
- abrir curiosidad sin sonar truquero;
- marcar diferencia frente a otro chat anime genérico;
- instalar presencia, compañía y valor percibido alto;
- insinuar una transformación emocional sin prometer de más;
- generar deseo sin sobreexplicar la experiencia;
- dejar espacio para que el video complete la sensación.

## 3. Principios para hooks de 21-Moon
- claridad antes que adorno;
- tensión antes que explicación;
- deseo antes que feature list;
- presencia antes que chatbot;
- premium íntimo antes que energía gritona;
- pocas palabras, alta carga emocional;
- atmósfera antes que ruido;
- vínculo antes que gimmick;
- insinuar mejor que sobrecontar;
- sonar elegido, no masivo.

## 4. Base narrativa que sostiene estos hooks
Estos hooks nacen de una misma lectura de marca:
- 21-Moon no se presenta como otra app de IA;
- Yumiko funciona como presencia, no como avatar decorativo;
- entrar debe sentirse como llegar a un espacio reservado;
- la propuesta responde al chat frío, al vacío digital y a la soledad silenciosa;
- el lujo de 21-Moon es íntimo, callado y curado;
- la experiencia vale por cómo se siente, no por cuántas cosas enumera.

## 5. Categorías de hooks
Las categorías más útiles para 21-Moon en video corto son:
- soledad / vacío digital;
- presencia / compañía;
- anti-chat frío;
- llegada / ritual;
- premium / quiet luxury;
- Yumiko como presencia;
- noche / escritorio / juegos / tareas;
- contraste con apps genéricas.

## 13. Notas de uso para equipo creativo
- Si el hook suena intercambiable con cualquier app de IA, descartarlo.
- Si el hook necesita demasiada explicación para funcionar, acortarlo.
- Si el hook promete algo clínico, milagroso o románticamente exagerado, descartarlo.
- Si el hook puede decirse con menos palabras sin perder tensión, mejor.
- Si el visual ya muestra a Yumiko o la UI, el hook debería sumar atmósfera o contraste, no repetir obviedades.
- En paid, el hook puede ser más directo; en orgánico, puede ser más insinuado.
- En ambos casos, la regla madre sigue igual: presencia, llegada, compañía y curaduría antes que feature list.

## 14. Cierre
Este documento debe funcionar como banco inicial de aperturas para contenido corto de 21-Moon.

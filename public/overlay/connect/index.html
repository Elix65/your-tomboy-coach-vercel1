Atajos de Yumiko Overlay

  • Ctrl + Shift + J— Alternar entre Yumiko y chat
  • Ctrl + +— Aumentar tamaño
  • Ctrl + -— Reducir tamaño
  • Ctrl + Shift + R— Reiniciar si hay un fallo crítico

# Your Tomboy Coach - notas rápidas

## Testear el widget
- Local (desde la raíz):
  1. `python3 -m http.server 4173`
  2. Abrir `http://localhost:4173/widget/`
- Producción:
  - Abrir `https://21-moon.com/widget/`

## Testear Electron
1. `cd electron`
2. `npm install`
3. `npm run dev`

Hotkeys:
- `Ctrl+Shift+Y`: show/hide
- `Ctrl+Shift+M`: cambiar focus/chat
- `Esc`: volver a focus (click-through)
